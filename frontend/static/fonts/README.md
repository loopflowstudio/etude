# Etude Fantasia fonts

These are the same files bundled by Loopflow's Swift application:

- Cormorant Garamond — display and section headings
- Lato — body and interface text
- JetBrains Mono — protocol identifiers and technical readouts

All three families are distributed under the SIL Open Font License 1.1; see
[OFL.txt](OFL.txt). The files are bundled locally so the certified play path
remains independent of remote font services.
